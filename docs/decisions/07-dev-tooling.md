# Feature: Developer Tooling

## Goal
Keep local development "splendid" with fast feedback and low setup friction.

## Tooling baseline
- `make`/`task` commands for common workflows
- `golangci-lint` for static checks
- unit/integration/e2e test targets
- local kind bootstrap command
- pre-commit hooks for formatting and quick validation

## Suggested DX commands
- `make test`
- `make test-e2e`
- `make run`
- `make docs-serve`
